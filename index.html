JSONauts

As a JSONaut1 you luv JSON no matter if used in the front-end, in the back-end or in APIs.

Are you a JSONaut?

Kick start

If you're not familiar with JSON (how comes?) of course you want to check out json.org first. Then, go and have a look at the RFC4627 where the application/json media type is defined.

In the front-end ...

JSON is used in jQuery, Dojo, backbone.js, etc.

In the back-end ...

JSON is used in MongoDB, CouchDB, elasticsearch, Node.js, etc.

APIs

There are over 2400 APIs listed on ProgrammableWeb.com that use JSON.
