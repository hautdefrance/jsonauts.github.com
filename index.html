JSONauts

The ultimate reference for JSON tooling and specs

As a JSONaut1 you luv JSON no matter if used in the front-end, in the back-end or in APIs. Now dive into all the JSON goodness and if you like what you see here consider to start contributing.

Basics

If you're not familiar with JSON of course you want to check out json.org first. Then, go and have a look at the RFC4627 where the application/json media type is defined.

In the front-end?

JSON is used in:

  • Backbone.js: model.toJSON(), collection.toJSON(), see also stackoverflow: [1], [2], [3]
  • Dojo: dojo/json
  • jQuery: getJSON(), parseJSON()

In the back-end?

JSON is used in:

  • CouchDB: HTTP Document API
  • elasticsearch: Index API
  • MongoDB: Writing Drivers and Tools
  • Node.js: Parsers, npm package ajax

In APIs?

There are over 2400 APIs listed on ProgrammableWeb.com that use JSON.

Contribute!

Wanna contribute? This site is hosted at GitHub as a GitHub Page so if you fancy extending the listing here simply send a pull request to jsonauts/jsonauts.github.com.

JSONaut is a wordplay: see Jason and the Argonauts.
